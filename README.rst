=======
Lapinoo
=======

Another raspberry appliance server developped using Janitoo


Needs
=====

- RFID
- screen
- RGB heart
- moving ears
- camera
- temperature / humidity
- proximity / light ambiance
- ....

Hardware
========

dht : air and humidity
gpio : servo and leds
spi : rfid + screen
i2c : proximity
...
